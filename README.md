# Full Stack Product Scraper App

## Architecture Overview

- Backend: NestJS (REST API)
- Frontend: Next.js (App Router)
- Database: PostgreSQL (or MongoDB)
- Scraping: Playwright
- CI/CD: GitHub Actions

Frontend communicates with Backend via REST APIs.
